# Company Logos

This directory contains company logo SVG files. Currently, these are placeholder SVGs with text-based logos.

## Replacing with Actual Logos

To replace these placeholders with actual company logos, you can download SVG versions from the following sources:

### Free Vector Logo Resources

1. **Worldvectorlogo.com** - https://worldvectorlogo.com/
   - Search for company names to find official logos
   - Download in SVG format

2. **Seeklogo.com** - https://seeklogo.com/
   - Extensive collection of company logos
   - Available in SVG, PNG, and other formats

3. **GetVectorLogo.com** - https://getvectorlogo.com/
   - Free vector logos in SVG and PNG formats

4. **VectorPortal.com** - https://www.vectorportal.com/
   - Free vector logos and graphics

5. **Wikimedia Commons** - https://commons.wikimedia.org/
   - Many company logos available under free licenses

### Important Notes

1. **Copyright**: Company logos are copyrighted and trademarked. Ensure you have the right to use them on your website.

2. **File Naming**: Keep the same filenames when replacing:
   - `tata-motors.svg`
   - `mercedes-benz.svg`
   - `liebherr.svg`
   - `putzmeister.svg`
   - `sandvik.svg`
   - `mahindra.webp`
   - `forbes-marshall.webp`
   - `aquarius-engineers.png`
   - `simem.png`
   - `linnhoff.png`

3. **SVG Optimization**: After downloading, you can optimize SVGs using:
   - https://jakearchibald.github.io/svgomg/
   - Or use `svgo` npm package

4. **Responsive Design**: The logos are displayed with:
   - Width: 200-280px
   - Height: 80px
   - Object-fit: contain

5. **Dark Mode Compatibility**: Consider using logos that work well in both light and dark themes, or provide separate versions.

## Alternative Approach

If you have direct access to company brand guidelines, check their official websites for downloadable logo packages:
- Look for "Press Kit" or "Media Resources" pages
- Many companies provide official logo files for partners

## Contact

If you're unable to find specific logos, contact the respective companies' marketing departments to request official logo files.
