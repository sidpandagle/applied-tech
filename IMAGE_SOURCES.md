# Image Sources for Applied Technologies Website

This document lists recommended free, high-quality images from Unsplash and Pexels for your manufacturing website.

## Instructions
1. Click the links below to open each image
2. Download the image (select appropriate resolution - 1920px wide recommended)
3. Rename the downloaded file to the suggested filename
4. Place it in the corresponding folder in `public/images/`

## Client Logos

For client logos, you should:
1. Search for official logos on the company websites
2. Use PNG format with transparent backgrounds when possible
3. Respect trademark guidelines
4. Or use text-based placeholders until you get official logos

### Recommended approach for logos:
- Visit each company's press/media page for official logo downloads
- Store in `public/images/clients/`
- Name format: `company-name-logo.png`

## Notes

- Download high-resolution images (1920px width minimum)
- Optimize images before uploading (use tools like TinyPNG or ImageOptim)
- Maintain consistent aspect ratios within each category
- Consider image file sizes for performance (aim for <500KB per image after optimization)
